Программа
вебинаров

Построение облачной инфраструктуры коммерческих и государственных предприятий

Space VM

Платформа серверной виртуализации для масштабируемых и надёжных частных облаков с уникальной технологией FreeGRID для работы с GPU NVIDIA без лицензий

Space VDI

Платформа виртуализации рабочих столов, обеспечивающая безопасный доступ к виртуальным и физическим рабочим местам из любой точки мира, с поддержкой российских операционных систем.

Space Cloud

Платформа управления мультиоблачной инфраструктурой с порталом самообслуживания, автоматизацией и биллингом, позволяющая объединять ресурсы локальных и внешних облаков

Space Client

Клиентское ПО для удобного доступа к виртуальным рабочим столам на Windows, Linux и macOS, включая плавный переход на VDI-инфраструктуру
